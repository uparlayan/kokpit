# Kokpit - Personal New Tab / Kişisel Yeni Sekme

[English](#english) | [Türkçe](#türkçe)

<a name="english"></a>
## 🇬🇧 English

**Kokpit** is a modern "New Tab" extension for Chrome and Firefox. It turns your browser's start page into a high-efficiency command center for managing multiple projects and Google accounts.

### ✨ Key Features
*   **🌐 Multi-Browser:** Supports **Chrome** and **Firefox** (MV3).
*   **🚀 Custom Grid:** Organize top sites in a visual grid.
*   **🖼️ Smart Favicons:** Multi-stage loading (Google S2 & DuckDuckGo).
*   **📂 Sidebar:** Hierarchical favorites (up to 3 levels).
*   **👥 Profiles:** Unique shortcuts for Work/Personal/Projects.
*   **🔑 Google AuthUser:** Quick switch between accounts (0-1-2) from sidebar.
*   **📝 Notes:** Color-coded quick notes.
*   **🛡️ Secure:** XSS-protected DOM (no innerHTML) per AMO standards.
*   **🛠️ Layout:** Draggable sidebar resizers.

### 🚀 Installation
- **Chrome:** Load unpacked folder in `chrome://extensions/`.
- **Firefox:** Load via `about:debugging` or install signed `.xpi`.

<a name="türkçe"></a>
## 🇹🇷 Türkçe

**Kokpit**, Chrome ve Firefox için modern bir "Yeni Sekme" eklentisidir. Tarayıcınızı çoklu proje ve Google hesabı yöneten kullanıcılar için tasarlanmış yüksek verimli bir komuta merkezine dönüştürür.

### ✨ Öne Çıkan Özellikler
*   **🌐 Çoklu Tarayıcı:** **Chrome** ve **Firefox** (MV3) ile tam uyumlu.
*   **🚀 Özel Izgara:** En çok kullandığınız siteleri görsel bir ızgarada düzenleyin.
*   **🖼️ Akıllı Favicon:** Çok aşamalı ikon yükleme (Google S2 & DuckDuckGo).
*   **📂 Yan Menü:** 3 seviyeye kadar favori ve klasör yönetimi.
*   **👥 Profiller:** İş, Kişisel vb. farklı bağlamlar için ayrı kısayollar.
*   **🔑 Google AuthUser:** Yan menüden hesaplar (0, 1, 2...) arası hızlı geçiş.
*   **📝 Notlar:** Renk kodlu hızlı not alma alanı.
*   **🛡️ Güvenli:** Firefox (AMO) standartlarına uygun, XSS korumalı DOM yapısı.
*   **🛠️ Esnek Yerleşim:** Sürükle-bırak ile ayarlanabilir yan menü genişliği.

### 🚀 Kurulum
- **Chrome:** `chrome://extensions/` üzerinden paketlenmemiş öğe olarak klasörü seçin.
- **Firefox:** `about:debugging` üzerinden geçici olarak veya imzalanmış `.xpi` ile kurun.
